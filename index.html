📖 사용 가이드

🏠 대시보드
전체 재고 현황을 한눈에 파악할 수 있는 메인 화면입니다.
• 총 입고 / 총 출고 / 현재 재고 / 설치 기기 수 통계 카드가 실시간으로 표시됩니다.
• 재고 부족 경보: 향기별 재고가 3개 이하로 떨어지면 자동으로 경보가 뜹니다.
• 최근 이력: 가장 최근 5건의 입출고 내역을 보여줍니다.
📥 입고 등록
향기 카트리지 또는 방향제 기기가 새로 입고됐을 때 사용합니다.
• 단건 등록: 한 종류씩 입력. 등록 즉시 시리얼 번호가 자동 발급되고 인쇄할 수 있습니다.
• 대량 등록: 여러 향기를 동시에 입력. 한 번에 여러 시리얼이 발급됩니다.
• 시리얼 형식: CT-202503-001 (카트리지), HC-202503-001 (기기)
📤 출고 등록
고객(약국)에게 제품을 출고할 때 사용합니다.
• 고객명 / USER-KEY / 설치 주소를 반드시 입력해야 자산 현황에서 추적이 가능합니다.
• 기기 출고 시 시리얼 번호를 입력하면 자산 현황에 자동 등록됩니다.
• 대량 등록: 여러 고객에게 동시 출고 시 행 추가로 일괄 처리합니다.
🗃️ 재고 현황
향기별 입고 / 출고 / 회수 / 잔여 재고를 확인합니다.
• 상단 월별 탭을 클릭하면 해당 월의 현황만 필터링해서 볼 수 있습니다.
• 잔여 재고 3개 이하는 빨간색으로 표시됩니다.
🗺️ 자산 현황
출고된 기기가 어느 고객(약국)에 설치돼 있는지 추적합니다.
• 회수: 기기를 다시 수거할 때 사용. 해당 기기 상태가 '회수'로 변경됩니다.
• 파손/불량 등록: 우측 상단 빨간 버튼. 파손·불량 기기를 출고 불가 상태로 관리합니다.
• 상태 필터로 설치중 / 회수 / 파손 / 불량을 구분해서 볼 수 있습니다.
🏷️ 시리얼 관리
발급된 모든 시리얼 번호를 조회하고 인쇄합니다.
• 필터로 재고 중 / 출고됨을 구분해서 볼 수 있습니다.
• 우측 상단 전체 인쇄 버튼으로 시리얼 라벨을 일괄 출력할 수 있습니다.
🧴 용품 재고
임직원이 서비스에 사용하는 소모품 재고를 관리합니다.
• 관리 품목: 바이러스케어 소독액 / 케어건 / 고글 / 케어존 스티커 / 완료 스티커
• 유형: 입고 (재고 추가) / 사용 (소비) / 파손·폐기
• 일괄 등록: 모든 품목 수량을 한 번에 입력할 수 있습니다.
• 재고 5개 이하 시 자동으로 '부족' 경보가 표시됩니다.
👕 유니폼 관리
임직원 유니폼(하복·동복·춘추복·배송조끼)을 사이즈별로 관리합니다.
• 유형: 입고 / 지급 (임직원에게 지급) / 반납 / 폐기·분실
• 일괄 등록: 종류 × 사이즈(L~6XL) 표에 한 번에 수량을 입력합니다.
• 임직원 이름으로 검색해서 누가 어떤 유니폼을 받았는지 확인할 수 있습니다.
📋 입출고 이력 / 🗑️ 데이터 삭제
• 입출고 이력: 모든 등록 내역을 유형별로 필터링해서 조회합니다. 각 행 우측 삭제 버튼으로 개별 삭제 가능합니다.
• 데이터 삭제: 잘못 입력된 데이터를 카테고리별로 찾아 삭제합니다. 삭제 후 복구는 불가능하니 주의하세요.
📊 Google Sheets 연동
입출고 등록 시 Google Sheets에 자동으로 데이터가 기록됩니다.
• 연동된 시트: 김포 MFC 향균케어 재고관리장
• 데이터는 월별 시트 탭으로 자동 분류됩니다. (예: 2025년 3월)
• 인터넷이 끊긴 경우 로컬에 저장 후 재연결 시 재고 현황 → Sheets로 내보내기로 수동 동기화 가능합니다.
